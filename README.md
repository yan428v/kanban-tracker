**Панель управления задачами команды**


Описание

Приложение для управления проектами и задачами команды.

Основные функции

- Доска задач (канбан) с колонками: «Запланировано», «В работе», «Готово»
- Создание и редактирование задач
- Возможность добавлять участников к задачам
- Статистика выполненных задач

Дополнительные функции

- Система комментариев в задачах
- Уведомления о новых задачах и изменениях
---

Требования к бэкенду


1. Модель данных для основных функций проекта

Минимальный набор таблиц:
Таблица задача: идентификатор, заголовок, дата, описание, тип, ссылка на автора, ссылка на список пользователей, статус

2. Модель данных для сервиса регистрации и аутентификации

Поля пользователя: идентификатор, имя, фамилия, логин, пароль

3. API для регистрации, авторизации и аутентификации

Эндпойнты: регистрация, аутентификация, смена пароля, обновление токена

4. API для управления задачами

Операции: создание, удаление, редактирование, чтение (с пагинацией, фильтрами по типу, дате, автору, статусу)
---

Результат реализации бэкенда

1. Разработана модель данных и настроены миграции
2. Создан API для работы с основными функциями. Код запускается и выполняет требования задания. Допускается и приветствуется любое расширение, дополнение и обоснованное улучшение функционала
3. Разработан набор тестов для проверки всех эндпойнтов API
4. Развёртывание: создан файл конфигурации Docker Compose для сборки и запуска проекта в контейнере


##### работа с alembic:
1. после изменения модели, генрируем миграцию
```bash
alembic revision --autogenerate -m "commit message"
```
3. проверяем сгенерированный файл в alembic/versions
4. применяем к бд
```bash
alembic upgrade head
```
- upgrade - применить миграции
- head - до самой новой миграции

для применения миграций в новом репе та же команда
